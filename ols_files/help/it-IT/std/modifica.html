Abilitazione della modifica dei dati di una collezione.

Premessa.

Il programma Ordiniamo/Let's Sort alias DDJ non è un DBMS bensì un programma per l'esplorazione e la manipolazione delle collezioni di dati, pertanto l'operatore, se lo desidera, ha piena facoltà di apportare modifiche ai dati delle Collezioni sotto la sua responsabilità.


La salvaguardia dell'integrità dei dati contenuti nelle collezioni viene lasciata alla responsabilità dell'operatore che si assumerà la responsabilità delle modifiche potenzialmente distruttive che deciderà di apportare alle collezioni di dati.


Come misura precauzionale, in ogni caso, le collezioni di dati vengono aperte (cioè caricate) per default in modalità Read Only [RO].


Le manipolazioni potenzialmente distruttive, che potrebbero compromettere l'integrità dei dati, richiedono una esplicita abilitazione da parte dell'operatore che così facendo se ne assume la responsabilità.

Abilitazione della modifica dei dati.

Per misura precauzionale, per default, il programma OLS propone sempre il caricamento delle collezioni di dati in modalità di sola lettura (Read Only [RO]),


Per cui se si accetta l'opzione di default la collezione sarà aperta in modalità di sola lettura (Read Only [RO]), la barra del titolo pertanto sarà colorata di rosso e riporterà la dicitura [RO].



Se si desidera operare sulle collezioni di dati allo scopo di correggere i dati è necessario abilitare le modifiche esplicitamente.

Commutazione da READ ONLY [RO] a Read Write [RW].

Per abilitare l'editing e la modifica dei dati è necessario cliccare sul bottone Select Collection della barra dei comandi, togliere la spunta alla casella Read Only [RO] e confermare premendo sul bottone Confirm.

La barra del titolo assumerà il colore celeste per segnalare il cambiamento di stato, inoltre la dicitura [RO] verrà sostituita da [RW].


Ripristino della modalità READ ONLY [RO].

Per ripristinare la modalità READ ONLY [RO] basta cliccare sul bottone Select Collection della barra dei comandi e confermare premendo sul bottone Confirm.
