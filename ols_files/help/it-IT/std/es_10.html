Esempio 10

Impiego dei Web Server in combinazione con OLS.

Il linguaggio JavaScript è stato espressamente progettato per l'automazione Web, cioè per comunicare con i siti Web per mezzo della rete caricando e scaricando informazioni.Sfortunatamente non sono tutte rose e fiori. Ogni sito web è organizzato in modo diverso; pochi prevedono la possibilità di scaricare i loro dati in formato CSV o equivalente, e quei pochi che lo fanno impongono che le richieste siano formulate in base ai loro protocolli. Nasce quindi il problema di implementare delle interfacce che permettano non solo il dialogo fra il programma di analisi dei dati e i server che li custodiscono, ma gestiscano in modo il più possibile uniforme la suddetta interazione.
Accolta l'idea di una simile struttura ci si potrebbe poi chiedere se non valesse la pena di prevedere la possibilità di utilizzare un web server per accedere in maniera controllata al file system del computer locale, ovviando alle limitazioni imposte dagli sviluppatori dei browser all'impiego di JavaScript per motivi di sicurezza.

Impiego di un web server per accedere ai file locali.

Accedere ai file locali per mezzo di un web server consente di centralizzare le richieste rivolte verso il file system per cui il controllo degli accessi risulta facilitato.
