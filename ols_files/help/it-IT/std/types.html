Tipi e Tracciato Record.

Il programma OLS prevede che i campi del tracciato record siano definiti facendo riferimento ai tipi registrati all'interno di OLS.

Tipi nativi di JavaScript.

JavaScript originariamente prevedeva sei tipi nativi: string, number, boolean, object , function e undefined. Definendo ES5 sono stati introdotti altri due tipi: symbol e bigint.
I tipi nativi di JavaScript costituiscono la base per la rappresentazione dei dati in memoria.
Ogni frammento di organizzazione deve obbligatoriamente essere codificato in riferimento ad uno dei tipi di dato nativi.

Tipi di HTML.

Per tipi HTML si intendono i tipi corrispondenti alle diverse modalità di input in HTML cioè ai diversi valori che possono essere assunti dall'attributo type dell'elemento <input> che sono:

button, checkbox, color, date, datetime-local, email,  file, hidden, image, month, number, password, radio, range, reset, search, submit, tel, text, time, url e week cui si aggiungono alcuni tipi corrispondenti a varie tipologie di elementi HTML come textarea, option-list eccetera.
