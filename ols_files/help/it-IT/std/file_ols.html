"_Hdr0_"

L'intestazione riporta in numero di release e l'informazione concernente la richiesta di eseguire automaticamente le funzioni di inizializzazione (AutoExec) al termine del caricamento della collezione.

"_aaFld_"

"_aColl_"

Elenco delle collezioni contenute nel file .OLS.

"autoexec"
