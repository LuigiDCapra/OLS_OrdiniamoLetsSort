Front End SQL (Sperimentale).

Prerequisiti.

  • L'impiego del programma Ordiniamo/Let's Sort con font end dell'installazione di MariaDB eseguita mediante XAMPP presuppone l'installazione del file rpc_exec.php sullo stesso server al fine di contentire la comunicazione fra OLS e il DBMS MariaDB.
  • L'utilizzo delle suddette funzioni presuppone la conoscenza di SQL da parte dell'operatore.
NOTA:
il funzionamento delle funzionalità considerate è sperimentale.

Scaricamento dei dati di una tabella mediante Input URL.

  • Cliccate sull'icona Load Collection della barra dei menu.

  • Selezionate Source URL.
  • Copiate nella casella URL l'indirizzo del server specificando il nome del DataBase e quello della Tabella desiderata.
    (ad esempio http://localhost/Relay/irc/rpc_exec.php?szNmDB=mysql&szNmTab=innodb_table_stats).
  • Selezionate Kind aObj.
  • Inserite il segno di spunta nella casella Embed.
  • Confermate.




La tabella innodb_table_stats del database mysql è particolarmente importante perché contiene l'elenco dei database e delle tabelle contenute nei diversi database.
