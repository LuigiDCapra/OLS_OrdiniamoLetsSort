Estrazione di una parte di una collezione.

Problema.

Considerata una generica collezione di dati in uno dei formati supportati dal programma DDJ ci si pone il problema di creare una nuova collezione costituita dai valori di alcuni campi dei record selezionati.


Ad esempio: data la collezione

{
"anatra_0":{"iNum1":41,"Bool2":false,"sz3":"duck_3","Date4":"2009-09-01","Str5":"S_asObj_5"},
"cane":{"iNum1":70,"Bool2":0,"sz3":"dog","Date4":"2021-01-02","Str5":""},
"cavallo":{"iNum1":53,"Bool2":true,"sz3":"horse","Date4":"2004-04-03","Str5":""},
"foca":{"iNum1":19,"Bool2":true,"sz3":"seal","Date4":"2000-10-04","Str5":""},
"gallo":{"iNum1":37,"Bool2":1,"sz3":"cock","Date4":"1988-08-05","Str5":""},
"gatto":{"iNum1":21,"Bool2":1,"sz3":"cat","Date4":"2017-02-06","Str5":""},
"lupo":{"iNum1":6,"Bool2":0,"sz3":"wolf","Date4":"2007-07-07","Str5":""},
"oca":{"iNum1":115,"Bool2":true,"sz3":"goose","Date4":"1996-06-08","Str5":""},
"rana":{"iNum1":84,"Bool2":false,"sz3":"frog","Date4":"2015-05-09","Str5":""},
"topo":{"iNum1":72,"Bool2":false,"sz3":"mouse","Date4":"2023-03-10","Str5":""}
}

Si potrebbe essere interessati solo ai valori dei campi iNum1 e sz3 dei record la cui data (Date-4) è più recente del primo gennaio 2025.

Per ottenere il risultato desiderato si dovrà impostare un filtro specificando la condizione concernente le date, modificare il tracciato record inibendo la visualizzazione dei campi indesiderati e quindi salvare la collezione dati risultate.

Impostazione del filtro.

Cliccate sulla voce Filter2 della barra dei menu, impostate la condizione


Date4 > 2015-01-01


usando i bottoni di scelta e confermate.

Risulterà selezionata la collezione:

{
"cane":{"iNum1":70,"Bool2":0,"sz3":"dog","Date4":"2021-01-02","Str5":""},
"gatto":{"iNum1":21,"Bool2":1,"sz3":"cat","Date4":"2017-02-06","Str5":""},
"rana":{"iNum1":84,"Bool2":false,"sz3":"frog","Date4":"2015-05-09","Str5":""},
"topo":{"iNum1":72,"Bool2":false,"sz3":"mouse","Date4":"2023-03-10","Str5":""}
}

Selezione dei campi cui si è interessati.

La selezione dei campi della collezione cui si è interessati deve essere fatta in negativo editando il tracciato record e inibendo la visualizzazione dei campi cui non si è interessati.

  • Volendo visionare o editare il tracciato record della collezione di dati selezionata bisogna
  • cliccare il bottone Select Collection della barra dei comandi,
  • selezionare Collection Loaded usando il menu a tendina Source;
  • selezionare Layout UsrView.aFld usando il menu a tendina Show Collection;
  • abilitare la modifica togliendo il segno di spunta alla casella Read Only [RO].
  • confermare premendo il bottone Confirm.
Per inibire la visualizzazione dei campi indesiderati basta togliere il segno di spunta corrispondente al campo fVisible nelle righe associate ai campi considerati e confermare.
Togliendo i segni di spunta ai campi Bool2, Date4, Str5 risulterà selezionata la seguente collezione:
{
"cane":{"iNum1":70,"sz3":"dog"},
"gatto":{"iNum1":21,"sz3":"cat"},
"rana":{"iNum1":84,"sz3":"frog"},
"topo":{"iNum1":72,"sz3":"mouse"}
}

Salvataggio della collezione risultante.

A questo punto per salvare la collezione risultante in un file esterno basterà cliccare sub bottone Save Collection della barra dei menu, impostare il nome del file in cui si desidera salvare la collezione, selezionando il formato e confermare.

La collezione verrà salvata in un file con il nome specificato nella directory di Download.

Salvataggio della collezione in ordine trasposto.

Nel caso in cui foste interessati a salvare la collezione risultante in ordine trasposto, scambiando cioè le righe con le colonne, vi scontrereste con il problema costituito dal formato della collezione.

La collezione

{
"cane":{"iNum1":70,"sz3":"dog"},
"gatto":{"iNum1":21,"sz3":"cat"},
"rana":{"iNum1":84,"sz3":"frog"},
"topo":{"iNum1":72,"sz3":"mouse"}
}

è costituita da un array associativo di oggetti, ma ora il  formato asObj prevede la memorizzazione dei valori degli elementi della collezione come come liste di coppie attributo valore separate da ':',

il suddetto vincolo impedisce il salvataggio della collezione di dati in ordine trasposto, pertanto se si desidera salvare la collezione in ordine trasposto sarà necessario cambiare formato selezionando ad esempio il formato aRcd e poi confermando la collezione verrà salvata con la rappresentazione seguente:

[
["cane","gatto","rana","topo"],
[70,21,84,72],
["dog","cat","frog","mouse"]
]
