Save Collection


La finestra di dialogo Save Collection consente di salvare la collezione selezionata esternamente sotto di forma di file in locale o in un sito remoto oppure all'interno del browser.

Parametri di salvataggio.

Destination

Il menu a tendina "Destination" consente di scegliere il supporto di memoria che si desidera utilizzare per il salvataggio dai dati contenuti nella collezione.
Sono previste le seguenti possibilità: file, local storage, session storage e URL.

File/Local Storage/Session Storage/URL.

Scrivere nella casella il nome del file in cui si desiderano salvare i dati.

Il file verrà salvato nella cartella (directory) di download impostato nel browser.

Save as:

Cliccando sul bottone di input si aprirà un menu a tendina che consente di scegliere il formato di salvataggio.


ATTENZIONE:

La selezione dei formati asRcd o asObj può causare la perdita di dati nel caso in cui più righe/tuple presentassero lo stesso valore nella colonna scelta come chiave, tipicamente la prima colonna.

Questo problema si presenta nel caso in cui nella collezione originale per l'accesso alla colonna richiedesse l'impiego di chiavi composte ottenute concatenando i valori di più attributi.

Index of the column used as Primary key

La casella consente di selezionare (quando è consentito) il numero della colonna da usare come chiave primaria.


Comandi della finestra di dialogo

X.

Cliccando sul simbolo "X" che appare nell'angolo in alto a destra si chiude la scheda annullando tutte le impostazioni.

Confirm.

Cliccando sul bottone "Confirm" disegnato nell'area verde in basso si chiude la scheda e si procede con il caricamento della Collezione selezionata.
