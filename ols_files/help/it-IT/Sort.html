Riordino dei valori di una Collezione.

Ordinamento dei valori di un campo cliccando sull'intestazione.


La vista Tabelle consente di riordinare l'elenco delle righe in funzione dei valori di un campo, cioè di una colonna, semplicemente cliccando sull'intestazione.

Cliccando una seconda volta sulla stessa intestazione la tabella verrà riordinata rispetto allo stesso campo in ordine inverso cioè in ordine decrescente se prima era crescente o vice versa.

Ordinamento Lessicografico (stringhe di caratteri).


Le stringhe di caratteri possono essere ordinate considerando le lettere maiuscole e minuscole uguali o diverse.

In inglese si dice che l'ordinamento e "Case Sensitive" se le stringhe:


"ciao", "CIAO", "Ciao", "ciAo"


si considerano diverse in caso contrario l'ordinamento è "NON Case Sensitive".


Il programma OLS consente di impostare la modalità di confronto fra le stringhe campo per campo impostando opportunamente il flag fCaseSens del Tracciato Record.

Per default il confronto è sempre Case Sensitive, per cui se si desidera considerare le stringhe di caratteri "ciao", "CIAO", "Ciao", "ciAo" come equivalenti bisogna disabilitare il flag.
