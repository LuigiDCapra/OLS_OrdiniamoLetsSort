Filter_QBE

delete

Clear the values of all entries (fields) in the card, so as to start again with an empty card.

copy

Save the values of the entries of the selected card in a buffer to allow them to be copied to another card if necessary.

paste

Initializes the fields of the selected card by filling in the corresponding entries saved in the "copy buffer".

Help file

Opens the context-sensitive Help screen, displaying information about the functions available in the given context.
